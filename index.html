Timr by jeena

A simple timer for OS X

Timr is a simple timer for OS X in a kind of a retro look. You can select the time by clicking on the digits, start it by clicking on the start button, which turns to pause, and in pause mode you can reset the timer by clicking on the little "C" which appears under the big start button.

Dependencies

OS X — Mac OS X

Install

Download it, unzip and move to the Application folder.
